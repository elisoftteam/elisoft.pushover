# elisoft.Pushover

A lightweight and simple .NET library for sending push notifications to mobile devices using the Pushover REST API.

✔ .NET 9  
✔ Dependency Injection friendly  
✔ ILogger support  
✔ Fully testable  
✔ Minimal dependencies  

---

## Installation

Install via NuGet:

```bash
dotnet add package Elisoft.Pushover
